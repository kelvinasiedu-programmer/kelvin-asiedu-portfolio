Selected Work

01

PulseCommerce

An ecommerce analytics platform built to turn store activity into dashboards, forecasts, churn signals, and experiment readouts that stakeholders can actually use.

Results: Transformed 450K raw ecommerce events into 5 analytics layers covering funnel diagnosis, demand forecasting, churn scoring, and A/B test evaluation.

Modeling & quality: Trained an XGBoost churn model to 0.889 AUC, authored 13 pytest cases covering 68% of the codebase, and containerized the platform with Docker.

Tech stack: Python, DuckDB, scikit-learn, XGBoost, Streamlit, Docker.

02

RAG Chatbot

A retrieval-augmented chatbot designed to produce grounded, context-aware answers instead of generic generation.

System design: Engineered an end-to-end RAG pipeline that combines FAISS vector retrieval with LLM generation for more reliable responses.

Iteration: Reduced hallucination through prompt engineering, embedding tuning, and response-evaluation loops, then shipped it as a FastAPI plus Next.js application.

Tech stack: Python, FastAPI, FAISS, Next.js, TypeScript, Docker.

Live Demo GitHub

03

Bank Account System

A banking system that models account types, tiered interest logic, and transaction safeguards through a fintech-style REST API.

Architecture: Built an OOP platform with Savings and Checking inheritance, 3% / 5% / 7% tiered interest rules, and overdraft validation.

Testing & delivery: Reached 38 passing unit tests across balance validation, transaction auditing, and account interactions, then exposed the system through FastAPI with Docker deployment.

Tech stack: Python, FastAPI, Pydantic, Docker, pytest.
